1. Project Objective
The aim of this project is to create a simple interactive image processing toolkit. The program allows the user to load an image, apply basic digital image processing operations, and compare the original and processed results visually.

2. Development Environment
The project was developed in Python. The graphical user interface was created with Tkinter. Pillow is used for image loading and display, OpenCV and NumPy are used for image processing operations, and Matplotlib is used to draw histograms.

3. Implemented Operations
The program supports image loading, original and processed image display, histogram display, point processing, histogram equalization, spatial filtering, thresholding, and several extra operations.

Point processing operations include negative transformation, log transformation, gamma transformation, contrast stretching, and bitplane slicing.

Histogram operations include displaying the histogram of the input image, displaying the histogram of the processed image, and histogram equalization.

Spatial filtering operations include mean filter, Gaussian filter, median filter, Laplacian filter, Sobel edge detection, Prewitt edge detection, and unsharp masking.

Thresholding operations include manual/global thresholding and Otsu thresholding.

Extra operations include Gaussian noise, salt and pepper noise, saving the processed image, and using the processed image as the new input.

4. Explanation of Methods
Negative transformation converts each pixel value using 255 - pixel value. Log transformation expands dark pixel values and compresses brighter ones. Gamma transformation changes image brightness depending on the gamma value. Contrast stretching expands the intensity range of the image. Bitplane slicing extracts one bit level from the grayscale image.

Histogram equalization improves image contrast by redistributing intensity values. Mean, Gaussian, and median filters are used for smoothing. Laplacian, Sobel, Prewitt, and unsharp masking are used for sharpening or edge detection. Manual thresholding uses a selected threshold value, while Otsu thresholding automatically finds a suitable threshold.

5. Conclusion
The project provides a small interactive laboratory environment for experimenting with basic digital image processing techniques. The user can load an image, select operations, change parameters, and visually compare the output with the original image.
